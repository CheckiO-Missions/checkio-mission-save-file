Sie erhalten eine Folge von Namen bestehender Dateien und einen einzelnen Dateinamen, der in der Folge gespeichert werden soll.

Wenn eine Datei mit demselben Namen bereits in der Sequenz vorhanden ist, sollte ein numerisches Suffix (n) an den Dateinamen angehängt werden (die erste ganze Zahl n, mit der ein neuer Dateiname in der Sequenz nicht vorhanden ist). Beachten Sie, dass der Dateiname bereits ein oder mehrere Suffixe enthalten kann.

Die Funktion sollte den Namen zurückgeben, unter dem die Datei in der Sequenz gespeichert wird.

Eingabe: Tupel von Strings (str{% if is_js %}ing{% endif %}) und einer Zeichenkette.

Ausgabe: Zeichenkette.
