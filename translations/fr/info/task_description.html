Vous disposez d'une séquence de noms de fichiers existants et d'un nom de fichier unique qui doit être enregistré dans la séquence.

Si un fichier portant le même nom existe déjà dans la séquence, un suffixe numérique (n) doit être ajouté au nom du fichier (le premier entier n, avec lequel un nouveau nom de fichier n'est pas présent dans la séquence). Notez que le nom de fichier peut déjà inclure un ou plusieurs suffixes.

La fonction doit renvoyer le nom sous lequel le fichier sera enregistré dans la séquence.

Entrée: Tuple de chaînes (str{% if is_js %}ing{% endif %}) et une chaîne.

Sortie: Chaîne.
